Hevy

This page explains how LiftShift uses Hevy data and what assumptions it makes.

Import options

  • CSV import: export your data from Hevy and upload it to LiftShift.
  • Sync methods: if you use a sync method, you still control what is accessed and stored locally.

What LiftShift reads

  • Exercise names
  • Set reps and weight
  • Dates / timestamps

Troubleshooting: date parsing

If LiftShift cannot parse workout dates, it is often caused by a non-English export format. Re-export in English and try again.
