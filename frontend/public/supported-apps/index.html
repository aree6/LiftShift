Supported apps

LiftShift supports importing data from the apps below. Each page explains the import method and expectations.

  • Hevy — CSV import and supported sync methods.
  • Strong — CSV import.
  • Lyfta — CSV import.

What LiftShift needs from exports

  • Exercise name
  • Date (and ideally time)
  • Reps and weight (when applicable)

Common import pitfalls

  • Non-English date formats or localized exports that change parsing.
  • Units embedded in headers (e.g., “Weight (kg)”).
  • Exercise names that vary across sessions.
